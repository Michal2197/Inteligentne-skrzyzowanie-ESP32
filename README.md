# Inteligentne skrzyżowanie ESP32

Projekt edukacyjny przedstawiający inteligentne skrzyżowanie wraz z przejazdem kolejowym wykonane z wykorzystaniem ESP32.

## Opis funkcjonalności

Projekt będzie symulował działanie inteligentnego skrzyżowania drogowego połączonego z przejazdem kolejowym.
Układ będzie automatycznie sterował sygnalizacją świetlną dla pojazdów zgodnie z zaprogramowaną logiką działania.
W momencie wykrycia nadjeżdżającego pociągu system uruchomi procedurę zabezpieczenia przejazdu kolejowego poprzez zmianę świateł drogowych oraz opuszczenie rogatek.
Rogatki będą sterowane za pomocą serwomechanizmów, które odwzorują rzeczywiste działanie przejazdu kolejowego.
Projekt będzie posiadał interfejs WWW umożliwiający zdalne sterowanie wybranymi funkcjami systemu oraz przeprowadzanie symulacji przejazdu pociągu.
Całość zostanie wykonana z wykorzystaniem mikrokontrolera ESP32.

## Wykorzystane elementy

- ESP32 DevKit
- serwomechanizmy
- diody LED
- rezystory
- przewody połączeniowe
- płytka stykowa
